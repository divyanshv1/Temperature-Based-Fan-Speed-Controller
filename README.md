# Temperature-Based Fan Speed Controller

## Overview
This project is an embedded system that automatically controls fan speed based on temperature using an LM35 sensor and microcontroller.

## Components Used
- Arduino / Microcontroller
- LM35 Temperature Sensor
- DC Fan / Motor
- Motor Driver (if used)
- Power Supply
- LCD Display (optional)

## Working
The LM35 sensor measures ambient temperature and sends an analog signal to the microcontroller. Based on the temperature value, PWM signals are generated to control the fan speed.

## Features
- Real-time temperature monitoring
- Automatic fan speed control
- Energy efficient system
- PWM-based motor control

## Applications
- Smart cooling systems
- Industrial temperature control
- Home automation

## Future Improvements
- IoT-based remote monitoring
- Mobile app control
